## 2단계 기능 구현 목록

### 📺 영화 상세정보 모달

- [ ] 영화 포스터 또는 제목 클릭 시 상세정보 모달 띄우기
- [ ] 모달에 영화 썸네일, 제목, 카테고리, 줄거리, 평균 평점 표시
- [ ] ESC 키 또는 닫기 버튼 클릭 또는 모달 바깥 클릭 시 모달 닫기
- [ ] 배경 블러 처리
- [ ] TMDB 상세정보 API 연동 (`/movie/{movie_id}`)

### ⭐️ 별점 매기기 기능

- [ ] 영화 상세 모달에서 내 별점 선택 UI 구현 (별 5개)
- [ ] 클릭한 별점 localStorage에 저장
- [ ] 새로고침 후에도 별점 유지
- [ ] 별점 점수에 따라 감성 문구 표시

  - 4점: 별로예요
  - 6점: 보통이에요
