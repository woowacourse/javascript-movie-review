# 기능 요구 사항

## UI

### 영화

- [x] 포스터 이미지가 없을 때, 대체 이미지 삽입

### 영화 상세 정보 조회

- [x] 페이지 로드되면 Modal 렌더링하고, 숨기기
- [x] MovieItem 클릭이 감지되면 Modal 보이게 하기
- [x] backdrop 클릭이 감지되면 Modal 숨기기
- [x] X 클릭/ESC 클릭이 감지되면 Modal 숨기기
- [x] 상세 정보 렌더링

### 별점 매기기

- [x] 별을 클릭하면 클릭한 위치만큼만, 색이 채워진 별이 렌더링
- [x] 별점에 따른 메세지 렌더링
  - [x] 0점: 별을 클릭해 영화를 평가해 주세요
  - [x] 2점: 최악이예요
  - [x] 4점: 별로예요
  - [x] 6점: 보통이예요
  - [x] 8점: 재미있어요
  - [x] 10점: 명작이예요
